# movie-industry-python-analysis
Exploratory data analysis of movie industry data using Python to identify factors influencing box office revenue.

# Movie Industry Data Analysis

## Project Overview
This project explores movie industry data to identify factors that influence a movie’s financial performance. The analysis focuses on variables such as budget, gross revenue, IMDb score, votes, genre, production companies, and country.
The goal of the project is to analyze relationships between these variables and identify patterns that affect a movie's commercial success.

## Tools Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset Description
The dataset contains information about movies including:
* Movie Title
* Genre
* Budget
* Gross Revenue
* IMDb Score
* Number of Votes
* Production Company
* Country
* Rating
* Release Information

## Data Cleaning and Preparation
Before conducting the analysis, several preprocessing steps were performed:
* Identified and handled missing values
* Removed duplicate records
* Converted budget and gross revenue columns to numeric data types
* Filtered rows with missing financial data


## Exploratory Data Analysis

### Budget vs Gross Revenue
A scatter plot and regression analysis were performed to examine the relationship between movie budgets and gross revenue.
The analysis shows a strong positive relationship between budget and gross revenue, suggesting that movies with higher production budgets often generate higher box office revenue.

### Correlation Analysis
A correlation heatmap was created to analyze relationships between numerical variables.
Strong positive correlations were observed between:
* Budget and Gross Revenue
* Votes and Gross Revenue

### Genre Analysis
Movies were grouped by genre to identify which genres generate the highest revenue.
The Action genre generated the highest revenue, followed by Comedy and Animation.

### Production Company Analysis
Production companies were analyzed based on total gross revenue generated by their movies.
Companies such as Warner Bros and Universal Pictures consistently produced high-performing movies.

### Country-wise Revenue Analysis
Revenue was also analyzed across countries to determine which countries generate the highest box office revenue.
The United States dominates the global movie industry in terms of revenue generation.

## Key Insights
* Movies with higher budgets tend to generate higher gross revenue
* Audience engagement (votes) is positively correlated with revenue
* IMDb rating has a weaker relationship with financial success
* Some genres consistently perform better at the box office
* Certain production companies dominate movie revenue generation
